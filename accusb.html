Акселерометр USB без микросхемы памяти

работает только будучи подключенным к USB компьютера, питание также подается по USB. После каждого измерения акселерометр передает результаты в компьютер.

На фото (вид сверху) между разъемом USB (слева) и микросхемой MEMS-акселерометра (справа) место для установки микросхемы памяти - в этом варианте она отсутствует.
Презентацию работы акселерометра: accelerometer.pps,

Акселерометр 16g USB и акселерометр 200g USB изготавливаются на одинаковых платах, их отличие в диапазонах измеряемых ускорений. Остальные параметры и программное обеспечение идентичны.

Прилагаемое программное обеспечение позволяет определить следующие параметры измерения:
- диапазон измеряемых ускорений

акселерометра 16g USB     ±2g,   ±4g,     ±8g или ±16g, 

акселерометра 200g USB ±25g, ±50g, ±100g или ±200g,

- число осей акселерометра, по которым выполняется измерение (использование не всех 3-х осей актуально при записи данных в энергонезависимую память, при построении графика по большому числу выборок - при построении графика по массивам данных в несколько мегабайт требуется ощутимое время, если информативны результаты, например, только по одной оси, время потсроения графика можно сократить втрое),
- частоту выборки акселерометра 3200 Гц, 1600 Гц, 800 Гц, 400 Гц, 200Гц, 100Гц, 50Гц, 25 Гц, 12,5Гц или 6,25 Гц
- время измерения / число выборок, ограниченные только объемом оперативной памяти вашего компьютера.

Программное обеспечение позволяет управлять акселерометром, считывать результаты измерения, отображать их в виде графиков ускорения и сохранять полученные данные в виде файлов.
Компьютерное приложение, работающее с акселерометром, обеспечивает удобные функции:

  1. просмотр графиков,
  2. обработку сигналов (сглаживание, вычисление спектра, тренда),
  3. поиск файлов по комментариям к ним,
  4. документирование результатов: 
  • сохранение данных графика или его фрагментов в текстовом формате txt или в формате csv (для импорта в MS Excel),
  • копирование в буфер обмена для полседующей вставки в документы, например, MS Word, PowerPoint,
    • графиков (в wmf-формат, качество картинки не теряется при сжатии/растягивании)
    • комментариев к файлам с автоматическим добавлением информации о файле (имя, время записи)
    • результатов вычисления спектра (значение и частота максимальной спектральной составляющей, все спектральные составляющие в текстовом формате) и тренда (формула, описывающая линию тренда)
В архиве acc-usb размещено демо-приложение со Справочной системой (Help-файл), в архиве dataAcc-usb - файлы данных, записанные акселерометром.

Для ознакомления со справкой можно прочесть файл в формате .rtf, размещенный в архиве.

Работа с графиком в приложении (увеличение/уменьшение масштаба, прокрутка) представлены в видео.

Обратите внимание на файл 03g_01g_005g_002g.axd, в котором записаны колебания по оси X с малыми ускорениями (0,3g 0,1g 0,05g и 0,02g), воспользовавшись функцией экспоненциального сглаживания для снижения шума, скрыв графики для осей Y и Z, вы сможете посмотреть, как выглядит малый сигнал на фоне шума, как снизить уровень шума за счет сглаживания.

Акселерометр USB с установленной на плате микросхемой энергонезависимой памяти 256КБайт

Акселерометр с памятью может работать автономно, питание от источника напряжения от +4,75В до +10В. Потребляемый ток 12мА.
Презентация акселерометра в автономном режиме:  autonomous.pps
работу акселерометра в автономном режиме также представляет видеоролик

Управление выбором  режима акселерометра выполняется по USB, затем акселерометр отключается от USB, измерения могут выполняться на объектах, на которые установка компьютера невозможна.

На левом фото в начале страницы представлена плата акселерометра с установленной микросхемой памяти 256КБайт - вы можете приобрести акселерометр и без нее, а при необходимости купить микросхемы памяти и стабилизатора, и установить их самостоятельно.

Энергонезависимость памяти акселерометра - хранение записанных в памяти данных не требует источника питания, запись и считывание данных требуют источника питания.

Способы запуска акселерометра в автономном режиме

запуск измерения может начинаться

- по команде, переданной по USB,
- по команде ПАУЗА-ЗАПУСК: через интервал времени, определяемый оператором, начнется измерение; за время паузы можно отключить устройство от USB компьютера, запустить процесс, параметры которого измеряются акселерометром. По окончании процесса акселерометр подключается к компьютеру для передачи записанных результатов измерния.
- по размыканию контактов на плате акселерометра:
для этого по USB передается команда, инициализирующая ожидание размыкания пары контактов на плате, после этого акселерометр можно отключить от компьютера. Теперь при размыкания контактов на плате акселерометра начнется измерение. К плате может подключаться внешнее устройство, например, с фотоприемником, обеспечивающее замыкание контактов при попадании светового луча на светочувствительный элемент.
Иллюстрированная  информация о режимах запуска и о подключении питания находится в Help-файле в разделе "Запись в микросхему памяти акселерометра"
(Help-файл в архиве acc-usb)

Прилагаемое программное обеспечение позволяет определить следующие параметры измерения:

- диапазон измеряемых ускорений

акселерометра 16g USB     ±2g, ±4g, ±8g или ±16g, 
акселерометра 200g USB ±25g, ±50g, ±100g или ±200g,

- число осей акселерометра, по которым выполняется измерение (использование не всех 3-х осей актуально при записи данных в энергонезависимую память, при построении графика по большому числу выборок - при построении графика по массивам данных в несколько мегабайт требуется ощутимое время, если информативны результаты, например, только по одной оси, время потсроения графика можно сократить втрое),

- частоту выборки акселерометра 3200 Гц, 1600 Гц, 800 Гц, 400 Гц, 200Гц, 100Гц, 50Гц, 25 Гц, 12,5Гц или 6,25 Гц

- время измерения / число выборок при записи в автономном режиме ограничены объемом памяти 256КБайт. Максимальное время записи в память  
        Tзаписи, секунд  =  (256 * 1024 - 20) /  2  / Число_осей  /  Частота_выборки,    
256 *1024 - объем памяти в байтах, 20 байт для хранения параметров измерения;
/ 2 - два байта хранят одну выборку;
Число_осей - от одной до трех (X, Y, Z) : например, если достаточно записать ускорение только по оси X, время измерения можно увеличить втрое.
Частота_выборки выбирается программно из приведенного выше ряда (3200Гц, 1600Гц, ..., 6,25Гц). Она выбирается в зависимости от того, насколько быстрые изменения сигнала требуется фиксировать. Подробнее об этом смотрите здесь: Частота выборки при ударном воздействии .
При записи по 3-м осям и частоте выборки 3200Гц максимальное время записи составит 13.6 секунды.

Акселерометр USB с памятью может передавать результаты и непосредственно в компьютер, без записи в микросхему памяти, тогда число выборок/время записи ограничивается лишь объемом оперативной памяти компьютера 

- только при записи в память можно выбрать старт по размыканию контактов, подключенных к палте акселерометра (например, кнопки, реле).

Обновление программного обеспечения акселерометра USB

При добавлении новых функций возможно обновление программного обеспечения как микроконтроллера, установленного на плате акселерометра USB, так и компьютерного приложения.

Архив acc-usb содержит последнюю демонстрационную версию компьютерного приложения - в демо-приложении можно просмотреть графики, протестировать обработку сигналов, записанных в файлы. Файлы для демонстрации работы приложения упакованы в архив dataAcc-usb

Как обновлять компьютерное приложение:
Последнюю рабочую версию приложения пользователи акселерометров USB могут запросить по e-mail.
 
ВНИМАНИЕ! Scales.scl - файл калибровочных коэффициентов вашего акселерометра - создается по результатам калибровки акселерометра и присылается при поставке акселерометра по e-mail вам в составе приложения! Сохраняйте этот файл, а при извлечении файлов из архива с обновлением заменяйте ВАШИМ файлом Scales.scl одноименный файл, извлеченный из архива.
ВНИМАНИЕ! Приложение для акселерометра USB не подходит для управления акселерометром с интерфейсом RS-485
Но возможности у обоих приложений одинаковы; формат записываемых этими приложениями файлов одинаков, поэтому файлы обоих типов акселерометров могут просматриваться и обрабатываться любым из этих приложений.

Обновления прошивки микроконтроллера:

Пользователи акселерометров USB могут запросить обновленную прошивку по e-mail

Как обновлять прошивку микроконтроллера на плате акселерометра:
-скачайте обновленную прошивку для микроконтроллера (ссылку на последний файл обновления *.blc смотрите ниже), запомните папку, в которую вы поместили файл обновления
-скачайте и распакуйте архив BootAccClient , запустите приложение BootAccClient.exe, следуйте указаниям, появляющимся на мониторе по мере выполнения приложения.
ВНИМАНИЕ! Загружать прошивку в микроконтроллер платы  акселерометра USB проще с того же компьютера, с которым работал акселерометр, так как для загрузки в микроконтроллер обновленной прошивки на компьютере должны быть установлены драйверы акселерометра USB!
